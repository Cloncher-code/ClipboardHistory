# ClipboardHistory

Менеджер буфера обмена для macOS, живущий в строке меню. Хранит историю скопированного (текст, картинки, файлы), позволяет искать, закреплять и быстро вставлять записи. Вдохновлён PasteNow.

## Возможности

- История буфера обмена: текст, форматированный текст (RTF), картинки и файлы
- Глобальный вызов панели по **⇧⌘V** из любого приложения, в том числе поверх полноэкранных
- Поиск, закрепление записей и пользовательские списки
- Автовставка по клику/Enter, навигация с клавиатуры (↑↓, Enter, цифры 1–9)
- Иконки приложений-источников, краткий и подробный режим отображения
- Звуки уведомлений на выбор, автоочистка по возрасту
- Пароли из менеджеров паролей в историю не попадают
- Оформление в стиле macOS 26 (Liquid Glass)

## Требования

- macOS 14 (Sonoma) или новее. Для полного стеклянного оформления — macOS 26.
- Xcode 16 или новее. Для стиля Liquid Glass — Xcode 26.

## Установка и запуск

1. Склонируйте репозиторий:
   ```
   git clone https://github.com/Cloncher-code/ClipboardHistory.git
   ```
2. Откройте файл **`ClipboardHistory.xcodeproj`** в Xcode (двойной клик).
3. Выберите свою команду подписи: верхняя строка проекта слева → вкладка **Signing & Capabilities** → поле **Team** → выберите ваш Apple ID (можно личный, бесплатный).
4. Нажмите **▶︎** (или ⌘R). В строке меню рядом с часами появится иконка 📋.

## Первый запуск

- Скопируйте что-нибудь (⌘C) — запись появится в панели. Откройте её кликом по иконке или по **⇧⌘V**.
- Для **автовставки** (когда клик по записи сразу вставляет её в активное окно) macOS попросит разрешение: **Системные настройки → Конфиденциальность и безопасность → Универсальный доступ** → включите тумблер напротив ClipboardHistory и перезапустите приложение.

## Примечания

- Приложение работает только из строки меню и не показывается в Dock.
- Для автовставки в настройках проекта отключена «песочница» (App Sandbox) — это уже сохранено в проекте, отдельно делать ничего не нужно.
- Чтобы приложение запускалось при входе в систему, включите соответствующий переключатель в настройках приложения (вкладка «Основное»).
