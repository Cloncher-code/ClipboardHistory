# ClipboardHistory

Менеджер буфера обмена для macOS, живущий в строке меню. Хранит историю скопированного (текст, картинки, файлы), позволяет искать, закреплять и быстро вставлять записи

## Установка 


1. Скачайте [ClipboardHistory_v1_5.zip](https://github.com/Cloncher-code/ClipboardHistory/releases/download/v1.5/ClipboardHistory.app.zip) и распакуйте. 
2. Перетащите ClipboardHistory.app в папку «Программы».
3. Кликните по приложению правой кнопкой → Открыть, затем в окне предупреждения ещё раз Открыть. (Обычный двойной клик на первый раз не сработает — это защита macOS для приложений не из App Store.)
4. Если macOS всё равно не пускает: Системные настройки → Конфиденциальность и безопасность, пролистайте вниз до сообщения о ClipboardHistory и нажмите «Всё равно открыть».
5. Для автовставки разрешите доступ: Конфиденциальность и безопасность → Универсальный доступ → включите тумблер напротив ClipboardHistory и перезапустите приложение.

## Возможности

- История буфера обмена: текст, форматированный текст (RTF), картинки и файлы
- Глобальный вызов панели по **⇧⌘V** из любого приложения, в том числе поверх полноэкранных
- Поиск, закрепление записей и пользовательские списки
- Автовставка по клику/Enter, навигация с клавиатуры (↑↓, Enter, цифры 1–9)
- Иконки приложений-источников, краткий и подробный режим отображения
- Звуки уведомлений на выбор, автоочистка по возрасту
- Пароли из менеджеров паролей в историю не попадают
- Оформление в стиле macOS 26 (Liquid Glass)

## Требования

- macOS 14 (Sonoma) или новее. Для полного стеклянного оформления — macOS 26.
- Xcode 16 или новее. Для стиля Liquid Glass — Xcode 26.
